# **App Name**: TipSync

## Core Features:

- Bill Details Input: Input the total bill amount, tip percentage, and number of people.
- Tip Display: Display the calculated tip amount per person.
- Round Up Feature: Allow users to specify whether the total amount should be rounded up.
- Restaurant Tip Feedback: Provide feedback, such as suggestions about different restaurants.
- Info Filtering: Allow the tool to reason whether certain information should be displayed, such as news or advertisements.

## Style Guidelines:

- Primary color: Soft purple (#B19CD9) for a friendly, inviting feel.
- Background color: Light pink (#F4C2C2) as a gentle, warm base.
- Accent color: Lavender (#E6E6FA) for highlighting important info, and contrasting nicely with the pink.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet approachable design.
- Use clean, line-based icons related to money and dining to illustrate amounts.
- Simple, intuitive layout for easy user input.
- Gentle transitions and feedback animations to enhance usability.
